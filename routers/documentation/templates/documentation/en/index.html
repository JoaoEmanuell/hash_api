Api Documentation

https://flaskhashapi.herokuapp.com/api

This api is a api able to generate hashes from values and compare the hashes generated by it!

It is necessary to use the method GET or POST to make requests from api

Generating a new hash

For generate a new hash access the api endpoint on the route generate and pass a parameter called value as query and as the value of this parameter pass a text that you want to use to create the hash


    GET https://flaskhashapi.herokuapp.com/api/generate/?value=valor
    
    GET https://flaskhashapi.herokuapp.com/api/generate/?value=HelloWorld

    POST curl -H "Content-Type: application/json" -X POST -d 
        '{"value" : "HelloWorld"}'
    https://flaskhashapi.herokuapp.com/generate/

If everything is right it will return a Json element containing a field called hash in the field is the generated hash.

Case the hash value is equal a Error means something went wrong during the hash generation!

Comparing hashes

To compare a hash access the api endpoint on the route compare, pass a parameter called value as query and as the value of this parameter pass a text that you want to use to create the hash, pass too other parameter called hash and as the value of this parameter pass the hash that was generate in generate


    GET https://flaskhashapi.herokuapp.com/api/compare/?value=valor&hash=hash

    GET https://flaskhashapi.herokuapp.com/api/compare/?value=HelloWorld&hash=872e4e50ce9990d8b041330c47c9ddd11bec6b503ae9386a99da8584e9bb12c4

    POST curl -H "Content-Type: application/json" -X POST -d '
        {"value" : "HelloWorld", 
        "hash" : "872e4e50ce9990d8b041330c47c9ddd11bec6b503ae9386a99da8584e9bb12c4"}' 
    https://flaskhashapi.herokuapp.com/api/compare/

If everything is right it will return a Json element containing a field called status this field contain a boolean value, being true if the value is corresponding a hash or false if the value is not corresponding a hash
