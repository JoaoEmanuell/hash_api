Documentação da api

https://flaskhashapi.herokuapp.com/api

Essa api é uma api capaz de gerar hash a partir de valores e comparar os hashes gerados por ela!

É necessário utilizar o método GET ou POST para fazer as requisições a api

Gerando um novo hash

Para gerar um novo hash acesse o endpoint da api na rota generate e passe um parâmetro chamado de value como query e como valor desse parâmetro passe o texto que você deseja utilizar para criar o hash


    GET https://flaskhashapi.herokuapp.com/api/generate/?value=valor
    
    GET https://flaskhashapi.herokuapp.com/api/generate/?value=HelloWorld

    POST curl -H "Content-Type: application/json" -X POST -d 
        '{"value" : "HelloWorld"}'
    https://flaskhashapi.herokuapp.com/generate/

Caso de tudo certo ela irá retornar um elemento Json contendo um campo chamado hash nesse campo está o hash gerado.

Caso o valor de hash seja igual a Error significa que algo deu errado durante a geração do hash!

Comparando hash

Para comparar um hash acesse o endpoint da api na rota compare, passe um parâmetro chamado de value como query e como valor desse parâmetro passe o texto que você deseja utilizar para criar o hash, passe também outro parâmetro chamado de hash e como valor desse parâmetro passe o hash que foi gerado em generate


    GET https://flaskhashapi.herokuapp.com/api/compare/?value=valor&hash=hash

    GET https://flaskhashapi.herokuapp.com/api/compare/?value=HelloWorld&hash=872e4e50ce9990d8b041330c47c9ddd11bec6b503ae9386a99da8584e9bb12c4

    POST curl -H "Content-Type: application/json" -X POST -d '
        {"value" : "HelloWorld", 
        "hash" : "872e4e50ce9990d8b041330c47c9ddd11bec6b503ae9386a99da8584e9bb12c4"}' 
    https://flaskhashapi.herokuapp.com/api/compare/

Caso de tudo certo ela irá retornar um elemento Json contendo um campo chamado status esse campo contem um valor booleano, sendo true caso o valor corresponda ao hash ou false caso o valor não corresponda ao hash
