Documentação da api

Acessar a api clicando aqui!

https://flaskhashapi.herokuapp.com/api

Essa api é uma api capaz de gerar hash a partir de valores e comparar os hashs gerados por ela!

É necessário utilizar o método GET para fazer as requisições a api

Gerando um novo hash

Para gerar um novo hash acesse o endpoint da api na rota generate e passe um valor como parâmetro, o hash será gerado a partir desse valor


    GET https://flaskhashapi.herokuapp.com/api/generate/{value}
    
    GET https://flaskhashapi.herokuapp.com/api/generate/HelloWorld

Caso de tudo certo ela irá retornar um elemento Json contendo um campo chamado hash nesse campo está o hash gerado.

Caso o valor de hash seja igual a Error significa que algo deu errado durante a geração do hash!

Comparando hash

Para comprar um hash acesse o endpoint da api na rota compare, passe o valor de texto para comparar e o hash equivalente a ele


    GET https://flaskhashapi.herokuapp.com/api/compare/{value}/{hash}

    GET https://flaskhashapi.herokuapp.com/api/compare/HelloWorld/872e4e50ce9990d8b041330c47c9ddd11bec6b503ae9386a99da8584e9bb12c4

Caso de tudo certo ela irá retornar um elemento Json contendo um campo chamado status esse campo contem um valor booleano, sendo true caso o valor corresponda ao hash ou false caso o valor não corresponda ao hash
